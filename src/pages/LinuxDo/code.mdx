本部分专门收录一些社区内常见的梗/黑话/暗语，以及一些社区内的特殊用语，帮助新人更快融入社区文化，更好地了解社区内的一些梗。如果有更多的梗，欢迎在 Wiki 的 官方GitHub 处留言，我们会及时更新！

### 佬站/佬友/热佬

<Callout type="info" emoji="">

L站 -> 佬站。始皇觉得用网友称呼大家不是非常贴切，既然叫 L站/佬站，干脆大家就以佬友互相称呼啦。所以热佬的也就是字面意思，指的是社区内的热心大佬。

但是只要你足够热心，遵循社区文化的**真诚、友善、团结、专业**，你也就是社区热佬的一员！

</Callout>

### aff 链接

<Callout type="info" emoji="">

AFF 是其他一些网站的邀请码，一般都会返佣金。aff 链接 简单来说就是带有返利推广的链接，因此带有此类链接的必须发布在 **一机难求** 板块，否则会被隐藏或者举报，详见论坛守则。

</Callout>

### 拼车、上车、来个车

<Callout type="info" emoji="">

GitHub Copilot 是一款由 GitHub 和 OpenAI 合作推出的人工智能编程助手工具，于2022 年 8 月 22 日开始收费。于是社区热佬写了一个小工具里面的功能就有 “上车、拼车” 这样的说法，解决了到处发用户ID拼车。如今，拼车、上车已经成为了社区内一个梗，用来表示可以一起分享免费的服务、资源等。

来不及解释了，快上车吧！

</Callout>

### 什么是C

<Callout type="info" emoji="">

信用卡诈骗是一种非法行为，即未经授权获取、贩卖或使用信用卡信息（通常用于购买礼品卡或预付卡）。信用卡诈骗会导致身份盗窃、个人和企业财务损失以及各种其他网络犯罪。

简单来说 C 就是一种`信用卡盗刷漏洞`，信用卡盗刷是一种严重的经济犯罪行为，不仅会给受害者造成巨大的经济损失和心理伤害，还会严重扰乱金融秩序，危害社会稳定。我们强烈呼吁大家远离此类违法行为，共同维护健康的金融环境。

**警告：国内的服务千万别C！因为真的有人会找上门！** 新人刚来了解了解即可，升级了去学习V佬的贴，前方的区域以后再来探索吧！

### 搞七捻三

<Callout type="info" emoji="">

“搞七捻三”，汉语词汇，[汉语拼音]gǎo qī niǎn sān，该词是吴语区的方言，属[吴语]用词。搞七捻三也可写作“搞七廿三”或“搅七捻三”，意思为胡闹、纠缠不清、说昏话。

搞七捻三 这个只能意会不能言传，总之就是好玩、搞笑、开心、舒服、爽！
