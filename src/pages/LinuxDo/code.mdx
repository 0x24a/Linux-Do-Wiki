### aff 链接

AFF 是其他网站的邀请码，一般都会返佣金，简单来说就是有返利推广的链接。因此带有此类链接的必须发布在 **一机难求** 板块，否则会被隐藏或者举报，详见论坛守则。

### 拼车、上车

GitHub Copilot 是一款由 GitHub 和 OpenAI 合作推出的人工智能编程助手工具，于2022 年 8 月 22 日开始收费。

于是社区热佬写了一个小工具里面的功能就有 “上车、拼车” 这样的说法，解决了到处发用户ID拼车。如今，拼车、上车已经成为了社区内一个梗，用来表示可以一起分享免费的服务、资源等。

来不及解释了，快上车吧！
