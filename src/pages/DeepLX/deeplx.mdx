## DeepLX 是什么?

Deeplx 是一个非官方的 DeepL API 实现，用于绕过 DeepL 的官方 API 限制或提供一些额外的功能。DeepL 是一家提供高质量机器翻译服务的公司，其官方 API 需要付费订阅。Deeplx 通过非官方的方法，提供类似的翻译功能。

## 如何获取 Deeplx API Key?

- 点击<Link href="https://connect.linux.do/" style={{ textDecoration: 'none', color: 'blue' }}>Connect.Linux.Do</Link>，获取你的专属 Deeplx API Key
