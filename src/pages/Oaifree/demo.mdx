## 旧版镜像 Pandora

兵马俑们，这里是满满的回忆啊！


### 功能说明

看这名字顾名思义，就是老版本的 Pandora。

此功能和新版几乎相同，只是不可以使用 GPTs，但是可以完美隔离对话和账号信息。

---

### 使用方法

 - 1. 通过始皇的<Link href="https://token.oaifree.com/auth" style={{ textDecoration: 'Access Token', color: 'blue' }}> Access Token </Link>服务获取 Access Token

 - 2. 使用 Access Token 在始皇的<Link href="https://chat.oaifree.com/token" style={{ textDecoration: 'Share Token', color: 'blue' }}> Share Token </Link>服务获取 Share Token（**可选**）

 - 3. 打开<Link href="https://demo.oaifree.com" style={{ textDecoration: 'none', color: 'blue' }}> demo.oaifree.com </Link>，填入 Access Token 或者 Share Token，即可使用！
